# Style Plan(it) Website

This is the official website for Style Plan(it), a personal styling consultancy based in Toronto.

## Project Overview

This project is a simple portfolio website built with vanilla HTML, CSS, and JavaScript. The content is managed dynamically through a CSV file.

For a more in-depth understanding of the project structure, design system, and technical details, please refer to the [project context document](ai-docs/context.md).
